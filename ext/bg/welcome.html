Welcome to Yomichan!

Here are some basics to get started

  • Clicking the Yomichan icon in the browser bar will open the quick-actions popup.
  • Clicking the cog button in the middle of the quick-actions popup will open the settings page.
  • Clicking the magnifying glass button on the left will open a search page, enabling text and terms to be looked up using the installed dictionaries. This can even be used in offline mode!
  • Holding the modifier key (Shift by default) as the cursor is moved over text will scan for definitions.
  • Clicking the speaker icon in a term's lookup results will play an audio clip of its pronunciation.
  • Clicking on an individual kanji in the term definition results will show additional information about that character. (Requires a kanji dictionary to be installed.)

Install dictionaries

Yomichan uses dictionaries for looking up terms, kanji, and more information. Dictionaries must be installed before Yomichan is able to scan text, which can be done using the controls on the right, or later from the the Settings page.

Several premade dictionaries can be found on the Yomichan homepage.
